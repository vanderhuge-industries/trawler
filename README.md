# Trawler

Trawler crawls awl.

## Installation

Add this line to your application's Gemfile:

    gem 'trawler'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install trawler

## Testing

(NOTE: Start yerself a mongo all up in your favourite way.)
